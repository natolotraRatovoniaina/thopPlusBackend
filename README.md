# API Backend Node.js - Auth & CRUD avec Sequelize

## Description
Cette application est une API backend développée avec Node.js et Express.
Elle implémente un système d’authentification, des opérations CRUD et une intégration avec une base de données via Sequelize.

Ce dépôt est une version publique d’un projet (en cours) initialement développé en environnement privé.
